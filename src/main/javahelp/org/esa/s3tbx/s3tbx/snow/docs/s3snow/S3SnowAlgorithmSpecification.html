SNAP Data Processors - Sentinel-3 Snow Properties Algorithm Specifications

Algorithm Specifications

The algorithms developed in the frame of the SEOM S3 Snow project and the SICE project are dedicated to the retrieval of snow properties such as snow spectral and broad band albedo, snow grain size, concentration of pollutants, and snow fractional coverage of the surface. These algorithms are mainly based on the solution of various sets of analytical equations for the different snow properties. All these methods are described in full detail in the corresponding project SEOM S3 Snow ATBD [3] as well as in a peer reviewed paper [5].
