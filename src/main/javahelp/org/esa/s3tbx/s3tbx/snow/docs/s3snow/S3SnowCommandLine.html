Sentinel-3 Snow - Command Line Description

Command Line Description

Since the Sentinel-3 Snow Properties processor makes use of the SNAP graph processing framework, it can be used also as a command line tool. The graph processing is invoked by the command

  • ${S3TBX-INSTALL-DIR}/bin/gpt

To obtain general help on the graph processing, use the command

  • ${S3TBX-INSTALL-DIR}/bin/gpt -h

Specific help on the OLCI Snow Properties processor can be obtained with

  • ${S3TBX-INSTALL-DIR}/bin/gpt -h OLCI.SnowProperties

Specific help on the OLCI SICE Snow Properties processor can be obtained with

  • ${S3TBX-INSTALL-DIR}/bin/gpt -h OLCI.SnowProperties.SICE

From these commands, information on the usage and a list of all available parameters are given.

The processor parameters can also be specified via graph xml files, e.g. olci_snow_graph.xml or olci_sice_snow_graph.xml . These graph xml files could look like the following. (Note that in principle it is not necessary to explicitly set parameters if default values shall be used). 


    <graph id="SnowPropertiesTest">
        <version>1.0</version>
        <node id="slope">
            <operator>OLCI.SnowProperties</operator>
            <sources>
                <sourceProduct>${sourceProduct}</sourceProduct>
                <cloudMaskProduct>${cloudMaskProduct}</cloudMaskProduct>
            </sources>
            <parameters>
                <ndsiThresh>0.04</ndsiThresh>
                <computePPA>true</computePPA>
            </parameters>
        </node>
    </graph>



    <graph id="SiceSnowPropertiesTest">
        <version>1.0</version>
        <node id="slope">
            <operator>OLCI.SnowProperties.SICE</operator>
            <sources>
                <sourceProduct>${sourceProduct}</sourceProduct>
                <brrProduct>${brrProduct}</brrProduct>
                <cloudMaskProduct>${cloudMaskProduct}</cloudMaskProduct>
            </sources>
            <parameters>
                <ndsiThresh>0.04</ndsiThresh>
                <considerNdsiSnowMask>true</considerNdsiSnowMask>
            </parameters>
        </node>
    </graph>

The graph for the OLCI Snow Properties processor can be invoked by the following call on the command line: 

    gpt <olci_snow_graph.xml> -SsourceProduct=<path-to-source-file>
    -ScloudMaskProduct=<path-to-cloudmask-file>

The call for the graph for the OLCI SICE Snow Properties processor would look like: 

    gpt <olci_sice_snow_graph.xml> -SsourceProduct=<path-to-source-file>
     -SbrrProduct=<path-to-brr-file> -ScloudMaskProduct=<path-to-cloudmask-file>
