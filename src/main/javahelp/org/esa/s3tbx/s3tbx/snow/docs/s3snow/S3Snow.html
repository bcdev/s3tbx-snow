SNAP Sentinel-3 Snow Processsor

The SNAP Sentinel-3 Snow Processor provides the retrieval of various snow properties (such as planar and spherical albedo for both spectral OLCI wavelengths and broadband ranges, snow grain diameter, snow pollution parameters, and others) from Sentinel-3 OLCI Level 1b input data.
