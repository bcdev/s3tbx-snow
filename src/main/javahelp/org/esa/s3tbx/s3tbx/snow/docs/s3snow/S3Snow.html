SNAP Sentinel-3 Snow Properties Processsors

The SNAP Sentinel-3 Snow Properties Processors provides the retrieval of various snow properties (such as planar and spherical albedo for both spectral OLCI wavelengths and broadband ranges, snow grain diameter, snow pollution parameters, and others) from Sentinel-3 OLCI Level 1b input data.

There are two distinct processors available:

  • The OLCI Snow Properties Processor, developed within the European Space Agency (ESA) Scientific Exploitation of Operational Missions (SEOM) Sentinel-3 for Science, Land Study 1: Snow, 2016-1018 [1].
  • The OLCI SICE Snow Properties Processor, developed within the pre-operational Sentinel-3 snow and ice products (SICE) project in 2019 [2], a direct evolution from the SEOM S3 for Snow project.

The OLCI SICE Snow Properties Processor reflects the latest and most settled state of the algorithms being developed in SICE. This processor is the recommended one for most users. The OLCI Snow Properties Processor reflects the state of the algorithms being developed and tested within SEOM S3 for Snow. This processor is, however, still of use for experienced users as it contains many user options to change specific algoritnm parameters as well as to generate additional bands in the target product.
