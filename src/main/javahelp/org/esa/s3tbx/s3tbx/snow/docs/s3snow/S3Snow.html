SNAP Data Processors - SNAP Sentinel-3 Snow Properties Processor

SNAP Sentinel-3 Snow Properties Processsor

The SNAP Sentinel-3 Snow Properties Processor provides the retrieval of various snow properties (such as planar and spherical albedo for both spectral OLCI wavelengths and broadband ranges, snow grain diameter, snow pollution parameters, and others) from Sentinel-3 OLCI Level 1b input data.
